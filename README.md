# Dockerized Web Application Deployment on AWS EC2

## Project Overview
This project demonstrates how to deploy a containerized web application on AWS EC2 using Docker and Linux. The application is packaged inside a Docker container and deployed on a cloud server.

## Architecture
User Browser
      ↓
AWS EC2 Instance
      ↓
Docker Container
      ↓
Nginx Web Server
      ↓
Web Application

## Technologies Used
- AWS EC2
- Docker
- Linux
- Nginx
- Git & GitHub

## Live Deployment

The application is deployed on an AWS EC2 instance using Docker containerization and can be accessed via the EC2 public IP.

## Project Setup

### 1 Launch EC2 Instance
Create an EC2 instance and allow the following ports in the security group:
- SSH (22)
- HTTP (80)
